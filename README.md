# SimpleShoppingList
Simple Shopping List application created with with Electron

###Version:
1.0.0

###USAGE
##Installation:
```sh
npm install
```

###SERVE
##To run electron app:
```sh
npm start
```

##PACKAGE AND BUILD
#WINDOWS
```sh
npm run package-win
```
#LINUX
```sh
npm run package-linux
```
#MAC
```sh
npm run package-mac
```
